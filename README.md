🤖 JARVIS AI: Discord Assistant
JARVIS es un asistente virtual diseñado para centralizar información y herramientas mediante comandos de voz y texto en Discord.

✨ Capacidades Clave
Monitoreo: Consulta el estado de tu hardware (CPU, RAM, Batería).

Información: Clima en tiempo real, noticias actualizadas y datos de Wikipedia.

Ciencia y Cultura: Acceso a imágenes de la NASA y base de datos cinematográfica (OMDb).

Productividad: Calculadora avanzada (WolframAlpha) y generador de códigos QR.

Interacción: Respuesta mediante voz sintética (TTS) en lenguaje español.

🛠️ Tecnología
Desarrollado en Python utilizando discord.py y una arquitectura asíncrona (aiohttp) para garantizar velocidad y eficiencia al consultar múltiples APIs.

📜 Comandos Rápidos
!estado | !clima [ciudad] | !noticias [cat]

!espacio | !mates [op] | !cine [título]

!wiki [tema] | !qr [enlace]
